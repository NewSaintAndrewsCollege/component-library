<p class="uk-text-lead">Create an animated loading spinner.</p>

## Usage

To create a spinner, add the `uk-spinner` attribute to a block element.

```html
<div uk-spinner></div>
```

{% include example.html content='
<div uk-spinner></div>
' %}

```html

```
